Image analysis and view tools

1. Image search

This tool allows to search for similar images in a local file directory for one target picture. It is similar to web reverse search, except it is done on your computer instead of the Internet. Web-image types are supported: jpeg, png, bmp, webp, gif.

First you select an image you want to search for. Then you select a folder where to search for similar images. The results are presented in a column of pictures ordered by similarity. The results also contain image sizes and file paths to the images. You can extract the paths as a list of strings by clicking "show/hide as list" link.

The search has an accelerator mode "Fast scan", which allows the program to use embedded thumbnails whenever available. This option is best used for newer generation unedited JPEG camera or phone images, which typically have properly formed thumbnails. "Fast scan" may miss older JPEG images or find incorrect similarity matches, because sometimes embedded thumbnails do not match the actual image.

2. Image clustering

Also called a duplicate finder, this tool allows to identify groups of similar images on disk. Those groups are called clusters. Web-image types are supported: jpeg, png, bmp, webp, gif.

You only need to select a folder where to search for similar images. The results are presented in a column of image clusters. The results also contain image sizes and file paths to the images. You can extract the clusters (groups) of image paths as a list by clicking "show/hide as list" link.

The clustering has an accelerator mode "Fast scan", which allows the program to use embedded thumbnails whenever available. This option is best used for newer generation unedited JPEG camera or phone images, which typically have properly formed thumbnails. "Fast scan" may miss older JPEG images or find incorrect similarity matches, because sometimes embedded thumbnails do not match the actual image.

3. JPEG embedded thumbnails reader

This tool allows to view embedded thumbnails, which are a part of some jpeg images. Apart of the full size image jpeg-files often contain a thumbnail for previewing the picture on the camera's LCD screen, in file managers, or in photo manipulation software.

You will select a folder to read files. The found thumbnails are presented in a column. The results also contain thumbnail sizes and image paths.

Not all jpeg-files contain thumbnails. Also not all, but typically the majority of thumbnails can be extracted with the tool.
